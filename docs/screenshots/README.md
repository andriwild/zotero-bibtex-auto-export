## Capture tips

- **Resolution:** capture at the native resolution of your display; don't upscale. Zotero 7 looks crisp on HiDPI screens, and the README's GitHub-flavored Markdown renders the images at their natural size up to the column width.
- **Theme:** use the system theme that gives the best contrast — typically light mode is more universally readable in screenshots, but the dark theme matches the rest of Zotero better. Pick one and stay consistent across all four shots.
- **Window chrome:** crop out the OS title bar and dock; the focus is the plugin UI, not your desktop.
- **PII:** before publishing, double-check the export path doesn't expose your home directory in a way you don't want public, and that no real-life library item titles in the collection picker are sensitive.
